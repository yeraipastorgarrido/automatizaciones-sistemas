# Scripts de Post-Instalación Ubuntu

Colección de herramientas en Bash para el aprovisionamiento rápido de sistemas basados en Debian/Ubuntu.

## Contenido Técnico
* **get_ready.sh**: Script de automatización para la configuración inicial de equipos de trabajo o servidores.

## Tareas Automatizadas
* **Actualización de repositorios**: Ejecución automática de `apt update` y `apt upgrade`.
* **Instalación de utilidades**: Despliegue de paquetes esenciales para la administración de sistemas (herramientas de red, editores y librerías base).
* **Optimización de entorno**: Configuración de elementos básicos del sistema para reducir el tiempo de puesta en marcha.
